1. HTML 기본 구조

```html
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>나의 HTML</title>
    </head>
    <body>
    </body>
</html>
```

- head 요소에는 문서제목, 문자코드 같이 해당 문서 정보를 담고 있으며, 브라우저에 나타나지 않는다.
- CSS 선언 혹은 외부 로딩 파일 지정은 가능하다.
- body요소에는 브라우저 화면에 나타나는 정보로 실제 내용에 해당한다.

2. 요소(element)
